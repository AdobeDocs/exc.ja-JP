# アセットを検索する{#search-for-assets}

Adobe Experience Cloudライブラリ内のアセットを検索して、組織全体または特定のフォルダー内のアセットを検索します。

検索バーを使用して、Experience Cloudライブラリ内のアセットを検索します。検索はファイル名を検索します。
