# アセットの検索{#search-for-assets}

Adobe Experience Cloud ライブラリ内のアセットを検索し、組織をまたいで、または特定のフォルダー内でアセットを見つけます。

検索バーを使用して、Experience Cloud ライブラリ内のアセットを検索します。検索では、ファイル名を探します。
